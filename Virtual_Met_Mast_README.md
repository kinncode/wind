# BSMI 虛擬測風塔 (Virtual Met Mast) 模型說明文件

本專案提供一套「虛擬測風塔 (Virtual Met Mast)」的機器學習解決方案，主要用於風場中 **100m 高度測風儀 (Anemometer) 的感測器備援與即時推估 (Imputation)**。

## 1. 專案目標與商業價值

*   **專案目標**：**不需要依賴時間延遲 (無預測未來)**，完全利用當下時間點 `t` 的低層風速感測器資料 (38m, 69m) 以及氣象特徵 (溫度、濕度、氣壓)，即時精準推估 `100m` 輪轂高度的風速。
*   **商業價值**：
    *   **感測器備援 (Redundancy)**：當高空的 100m 感測器因結冰、鳥擊、雷擊或定期維護而導致數據中斷或異常時，本系統能即時接手，提供高度準確的風速推估值。
    *   **確保控制系統穩定**：防止錯誤的風速讀值干擾風機的 Pitch Control 或發電量評估，確保 SCADA 系統持續接收穩定資料。

## 2. 檔案說明

*   `BSMI_Virtual_Met_Mast.py`：主要的機器學習訓練與推估腳本。
*   `models_virtual_mast/`：腳本執行後自動建立的資料夾，存放訓練好的模型與視覺化圖表。

## 3. 特徵工程設計

為了嚴格模擬「100m 數據遺失」的情況，模型訓練中**完全排除了所有與 100m 相關的變數**。

*   **目標變數 (Target)**：
    *   `WS_100_mean`：100E 與 100W 的平均風速 (作為 Ground Truth)。
*   **輸入特徵 (Features)**：
    *   **低層風速**：`WS_69W`, `WS_38W`
    *   **低層風向**：`WD_35` 經過三角函數轉換的 `WD_35_sin`, `WD_35_cos` 
    *   **環境氣象**：氣溫 `AT_95`、相對濕度 `RH_95`、氣壓 `BP_93`
    *   **空間動量衍生特徵**：
        *   `shear_69_38`：69m 與 38m 的絕對風切。
        *   `shear_69_38_ratio`：69m 與 38m 的風切比例。
    *   **時間統計特徵 (近 10 分鐘)**：
        *   `WS_69W_std_10m`, `WS_38W_std_10m`：利用過去 10 分鐘的標準差作為紊流強度的代理指標。
        *   `AT_95_diff_10m`：近期溫度變化率。
    *   **週期性特徵**：加入一日與一年的 Sin/Cos 編碼，捕捉大氣穩定度與季節性日夜變化。

## 4. 模型與訓練方式

*   **演算法**：使用 **LightGBM (LGBMRegressor)** 進行迴歸預測。
*   **驗證機制**：採用 5-Fold Cross Validation (五折交叉驗證) 確保模型沒有過度擬合 (Overfitting)。

## 5. 執行方式與輸出結果

### 執行方法
確保環境安裝好 `pandas`, `lightgbm`, `scikit-learn`, `matplotlib` 等套件，然後在終端機執行：
```bash
python BSMI_Virtual_Met_Mast.py
```

### 模型驗證結果
在 270 萬筆 (約 5 年) 的 1 分鐘解析度資料上，模型展現出極為驚人的精準度：
*   **R² (決定係數)**: `0.9960`
*   **RMSE (均方根誤差)**: `0.3614 m/s`
*   **MAE (平均絕對誤差)**: `0.2554 m/s`

### 產出檔案清單 (於 models_virtual_mast 目錄下)
1.  **`Virtual_Met_Mast_LGBM.pkl`**：訓練完成的最終模型，可直接載入 (使用 `joblib`) 進行即時預測。
2.  **`virtual_mast_scatter.png`**：推估風速 vs 真實風速的散佈對比圖。

4.  **`feature_importance.png`**：模型特徵重要性圖表 (決策貢獻最大的通常為 `WS_69W`)。
