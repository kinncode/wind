# 實驗四：大氣穩定度預測系統 (Atmospheric Stability Forecast System)

本專案實作大氣穩定度（以風速剖面冪律指數 $\alpha$ 表徵）的時間序列預測系統，預測未來 1 至 120 分鐘的穩定度軌跡與穩定度轉換警報。

## 1. 實驗目標與商業價值
* **大氣穩定度物理意義**：
  * $\alpha < 0.20$：不穩定/中性大氣（對流混合強烈，紊流高，垂直風切小）。
  * $\alpha \ge 0.20$：穩定大氣（大氣層結穩定，垂直風力梯度大，風機葉片承受不均勻剪應力）。
* **商業價值**：提前預知大氣穩定度的變化（如 Not-Stable $\rightarrow$ Stable），能讓風場控制系統提前調節風力發電機組的 Pitch 角與轉速，防止劇烈的風剪剪應力造成風機機械疲勞與壽命減損，同時優化發電效率。

## 2. 核心技術改進 (v1 vs. v2)
本系統經歷了兩個版本迭代，在 v2 引入了時序預估的黃金法則：
1. **多尺度時間序列特徵工程**：捨棄 v1 的單純歷史點數展平，v2 從過去 120 分鐘時序中提取了 **71 維特徵**（包括多尺度滾動均值/標差/峰度/偏度、一階與二階差分、EWMA、風向與氣象共變量、以及基於滾動 60 分鐘的 **FFT 頻域主週期與低頻能量比**）。
2. **Horizon-as-Feature (HaF)**：將預報距離 $h$（1–120 分鐘）直接作為模型的一個輸入特徵，僅需訓練一個統一模型，即可預測未來任意時間點。
3. **Expanding Window 交叉驗證 + Embargo**：採用嚴格的時序遞增切分驗證（5-Fold），並加入 120 分鐘的 embargo 緩衝期以徹底防止未來 Target 洩漏。

## 3. 實驗結果

### 5-Fold Expanding Window 驗證表現 (v2 最終模型)
在嚴格的時序交叉驗證下，模型表現高度穩定：

| 驗證折數 (Fold) | RMSE | MAE | $R^2$ | Accuracy (Acc) | Macro F1-Score |
|---|---|---|---|---|---|
| **Fold 1** | 0.1086 | 0.0439 | 0.6951 | 93.92% | 0.9105 |
| **Fold 2** | 0.0984 | 0.0388 | 0.6990 | 94.48% | 0.9107 |
| **Fold 3** | 0.0965 | 0.0369 | 0.7137 | 94.30% | 0.9234 |
| **Fold 4** | 0.1002 | 0.0392 | 0.7184 | 94.59% | 0.9222 |
| **Fold 5** | 0.0976 | 0.0372 | 0.6893 | 94.39% | 0.9075 |
| **平均值** | **0.0999** | **0.0392** | **0.7011** | **94.34%** | **0.9149** |

*註：相較於舊版 v1 在遠距離（如 t+120min）的 $R^2$ 衰退至 0.225，v2 模型在時序滾動特徵加持下，全預報區間的平均 $R^2$ 提升到了 **0.7011**，展現極強的魯棒性。*

## 4. 檔案清單
* `BSMI_Stability_Forecast_CPU_1_local.py`：大氣穩定度預報與預警之核心 Python 腳本。
* `BSMI_Stability_Forecast_CPU_1.ipynb`：Colab 或 Jupyer Notebook 實驗版本。
* `BSMI_Stability_Forecast_CPU_1_local.ipynb`：本地 Jupyter 實驗與繪圖版。
* `models/`：存放訓練好的穩定度預估模型（`.pkl` 格式，已排除版控）與各版本圖表：
  * `v1/`：初代模型的評估 CSV 與混淆矩陣圖等。
  * `v1_alt/`：初代的微調版模型（v1-1）。
  * `v2/`：導入 HaF 與時序特徵工程後的最終報告（`cv_evaluation_report.csv`、穩定度分布圖 `stability_distribution.png`、預報技巧衰減圖 `forecast_skill_decay.png`、預測軌跡圖 `forecast_trajectories.png`）。
