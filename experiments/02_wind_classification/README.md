# 實驗二：風況四級分類 (Wind Regime Classification)

本實驗使用隨機森林分類模型（Random Forest Classifier），將風力發電運轉狀態分為四個典型風況級別。

## 1. 實驗目標與商業價值
* **風機運行狀態預警**：風力發電機在不同風速下有不同的運行模式（例如：低風切入發電、額定滿載發電、強風切出停機等）。
* **風況級別定義**：
  * **低風** (Low Wind): $WS < 4$ m/s (低於切入風速，無法發電)
  * **正常** (Normal Wind): $4 \le WS < 10$ m/s (一般發電區間)
  * **強風** (Strong Wind): $10 \le WS < 20$ m/s (額定滿載發電區間)
  * **極端** (Extreme Wind): $WS \ge 20$ m/s (接近或超過切出風速，需停機保護)

## 2. 特徵工程設計
將原始 1 分鐘資料 resample 至 10 分鐘，並排除 100m 東側平均風速 (`WS_100E_mean`)，建立以下特徵：
* **多高度觀測**：`WS_100W` (西側同高度風速)、`WS_69W` (69m 風速)、`WS_38W` (38m 風速)
* **時序變化度**：100m 東側的滾動標準差 (`WS_100E_std`)、最大值 (`WS_100E_max`)、最小值 (`WS_100E_min`) 及變化範圍 (`WS_100E_range`)
* **亂流與風切**：低層亂流強度 (`TI_69W`, `TI_100E`)、風切指數 (`shear_38_69`, `shear_69_100`)、風速比 (`ratio_69_38`, `ratio_100_69`)
* **風向向量與氣象**：`WD_97_sin/cos`、`WD_35_sin/cos`、溫度 (`AT_95`)、氣壓 (`BP_93`)、濕度 (`RH_95`)
* **時間週期**：日與月的週期 sin/cos 編碼

## 3. 模型設定
* **演算法**：隨機森林分類器 (RandomForestClassifier, 300 棵樹, `max_depth=20`, `class_weight='balanced'`)。
* **資料切割**：80% 訓練集，20% 測試集，使用分層抽樣（Stratified Split）維持類別比例平衡。

## 4. 分類評估結果

模型在測試集上的表現極其優異，**總體準確率 (Accuracy) 達 99.62%**，F1-Score (加權) 達 99.62%。
