# 實驗三：風況分群分析 (Wind Condition Clustering)

本實驗使用非監督式學習（Unsupervised Learning），對風場的多維氣象特徵進行分群，藉此自動識別典型的風場氣候模式（Weather Regimes）。

## 1. 實驗目標與商業價值
* **風氣候模式識別 (Climatology Analysis)**：風場的發電效益與風力分布高度相依於宏觀氣象模式。透過無監督分群，能在大數據中自動找出特徵截然不同的季風與天氣型態，有助於精準預估發電產量和規劃風機運轉策略。

## 2. 特徵工程與降維
* **分群特徵**：
  * **三高度風速均值**：`WS_100E`, `WS_69W`, `WS_38W`
  * **大氣亂流與風切**：`TI_100E` (100m 亂流)、`TI_69W` (69m 亂流)、`shear_38_69` (低層風切指數)
  * **風向向量**：`WD_97_sin`, `WD_97_cos`
  * **大氣變量**：氣溫 (`AT_95`)、氣壓 (`BP_93`)
  * **日週期**：`hour_sin`, `hour_cos`
* **數據降維 (PCA)**：使用主成分分析（Principal Component Analysis），保留累計解釋變異量達 **95%** 的主成分，消除特徵共線性並降低噪訊。

## 3. 分群設定
* **演算法**：K-Means 聚類演算法。
* **最佳群數選擇**：利用 Elbow Method (肘部法) 搭配 Silhouette Score (輪廓係數) 在 $k=2 \sim 10$ 之間進行搜尋。為確保分群具備足夠的物理細緻度，最終選擇次佳輪廓係數之 **$k = 4$** 作為分群數。

## 4. 四大天氣模式分群結果

根據聚類中心的統計特徵，模型自動定義了 4 個完全符合台灣海峽風場物理特性的天氣型態：

### 物理意義分析
1. **東北季風效應 (群 1 與 群 3)**：
   * **群 3** 為典型的**冬季強勁東北季風**，伴隨極高的風速 (平均 16.39 m/s)、極低的溫度 (18.6°C) 與極高的氣壓 (1004.7 hPa)，這是風場的主要發電來源。
   * **群 1** 為**中等強度的東北風**，風速溫和 (平均 6.40 m/s)，多出現在春秋季過渡期。
2. **西南氣流效應 (群 2)**：
   * **群 2** 屬於**夏季西南氣流**，風速溫和 (5.71 m/s)，伴隨高溫 (27.0°C) 與低氣壓 (997.0 hPa)。
3. **夏季微風對流 (群 0)**：
   * **群 0** 為風場最弱風的時期 (平均 3.02 m/s)，高溫且風切呈負值（可能存在強烈的垂直對流混合或局部環流），亂流強度顯著偏高 (19.8%)。
